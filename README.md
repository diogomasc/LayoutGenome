# LayoutGenome

Landing page estática com foco em apresentação do produto LayoutGenome.

O projeto atual contém apenas frontend (HTML, CSS e JavaScript), sem fluxo ativo de extração no próprio site.

---

## Acesso Online

A landing page pode ser acessada em:

- https://diogomasc.github.io/LayoutGenome/

---

## Estrutura do Projeto

- `index.html`: estrutura principal da landing page.
- `css/`: estilos globais, componentes, layout e estilos da página inicial.
- `js/`: scripts de interação visual da landing (menu, FAQ, scroll, galeria e afins).
- `assets/`: mídias e arquivos estáticos.

---

## Execução Local

Como é um projeto estático, você pode abrir diretamente o `index.html` no navegador.

Opcionalmente, rode com um servidor local para desenvolvimento:

```bash
npx serve .
```

Depois acesse o endereço exibido no terminal (geralmente `http://localhost:3000`).

---

## Observação

Nesta versão, a landing está focada em experiência visual e conteúdo de produto.
Integrações de backend e geração automatizada de arquivos não fazem parte do fluxo ativo da página publicada.
